## 타입스크립트 기본 타입
---
타입스크립트의 주요 원시 타입은 모두 소문자이다. **String (X)** **string (O)**

### number
---
정수, 실수형에 따른 타입은 없으며 숫자는 모두 **number** 타입이다.
```ts
let num : number = 0;

num = 1
num = 5.2
num = 10
```

### string
---
- 일반적인 문자 및 문자열 타입이다.
```ts
let str : string = '';

str = 'Hi'
str = "Hi"
str = `Hi`
```

### Boolean
---
- 일반적인 Boolean 타입이다.
```ts
let check : booelan = false;

check = true
check = false
```
