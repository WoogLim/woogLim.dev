## 시작
---
이전 시간에는 var, let, const 키워드의 소개 `스코프`, `호이스팅`개념에 대해 설명했습니다. 이제 `변수`에 대해 빠르게 짚고 넘어가봅시다.

### 네이밍 컨벤션
---
변수의 식별자의 경우 아래 코드와 같이 네이밍 컨벤션을 사용합니다. 일반적인 변수나 함수 이름에는 카멜 케이스를 사용하고, 생성자 함수, 클래스 이름에는 파스칼 케이스를 사용합니다.  
ECMAScript사양에 정의된 객체, 함수들도 카멜 케이스와 파스칼 케이스를 사용합니다.
```js
// 카멜 케이스
var firstName;

// 스네이크 케이스
var first_name;

// 파스칼 케이스
var FirstName;
```
