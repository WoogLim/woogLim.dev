### 살펴보기
이번에는 컨테이너간 어떻게 서로간 통신이 가능한지 알아봅시다.

#### 기본
- 컨테이너를 실행하면 기본적으로 특정 Docker에 연결하는 백그라운드에 존재합니다. 바로 가상 프라이빗 가상 네트워크 브리지 네트워크에 존재합니다.
- 각 가상 네트워크는 호스트 IP의 NAT 방화벽을 통해 라우팅됩니다.
- 모든 컨테이너는 서로 통신하려는 특정 컨테이너가 있는 경우 실제로 `-p`옵션을 사용할 필요는 없습니다.
  - 예로 후속 서버, PHP Apache 컨테이너가 있는 애플리케이션이 있는 경우 두 컨테이너는 동일한 네트워크에 있어야 합니다.
